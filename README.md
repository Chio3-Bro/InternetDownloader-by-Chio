Описание
Download Manager — это настольное приложение для управления загрузками файлов из интернета. Оно позволяет контролировать процесс скачивания, ставить загрузки на паузу, возобновлять их и отслеживать прогресс в реальном времени. Все данные о загрузках сохраняются между сеансами работы.

Основные возможности
Управление загрузками: запуск, пауза, возобновление и отмена загрузок

Мониторинг прогресса: отслеживание скачанных/общих байтов и процента выполнения

Многозадачность: поддержка нескольких одновременных загрузок

Управление файлами: переименование и удаление скачанных файлов

Персистентность данных: автоматическое сохранение истории загрузок в JSON

Асинхронная архитектура: загрузки выполняются в фоновых потоках без замораживания UI

Технологический стек
.NET Framework / .NET 6+

Windows Forms — пользовательский интерфейс

HttpClient — HTTP-запросы

System.Text.Json — сериализация данных

Async/Await — асинхронные операции

SemaphoreSlim — синхронизация потоков для паузы/возобновления
